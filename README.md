# reqs-builder

YAMLメタデータからFP法で計測可能な要件定義書を生成するツール

## 概要

要件定義作業を省力化するツールです：

- ERD、DFD、CRUDなどの構造化されたメタデータをYAML形式で管理
- 自動整合性チェック
- Markdown + Mermaid.js形式でドキュメント生成

## 目的

ERD、DFD、CRUDマトリクスなどの要件定義成果物は、本質的に構造化されたデータの集まりです。
図表間の整合性を手作業で維持する代わりに、メタデータとして管理し、
整合性の取れたFP法計測可能な仕様書を生成します。
